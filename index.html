☁️ 클라우드 DB 영구 보존 설정

Supabase(무료 클라우드 PostgreSQL)에 연결하면 모든 데이터가 클라우드에 자동 저장됩니다. 브라우저/PC를 잃어도 URL과 키만 있으면 어디서나 같은 데이터로 복구됩니다.

📘 Supabase 5분 설정 가이드

  1. supabase.com 접속 → "Start your project" → GitHub/Google로 무료 가입
    신용카드 불필요. 무료 플랜은 500MB 저장 / 무제한 사용기간.
  2. "New Project" 클릭 → 이름(예: my-erp), 비밀번호(아무거나, 메모만), 지역(Northeast Asia/Seoul 권장) → 2분 대기
  3. 좌측 메뉴 "SQL Editor" 열기 → "New query" → 아래 SQL을 그대로 붙여넣고 "Run": 
    create table if not exists erp_data (
  workspace text primary key,
  data jsonb not null,
  updated_at timestamptz default now()
);
alter table erp_data enable row level security;
create policy "allow all" on erp_data for all using (true) with check (true);
  4. 좌측 메뉴 "Project Settings" → "API" 열기
    - Project URL 복사 → 위쪽 입력란에 붙여넣기
    - Project API keys → anon public 키 복사 → 위쪽 입력란에 붙여넣기
  5. 위 "🔌 연결 테스트 & 저장" 버튼 클릭 → 성공 메시지 확인 → 끝!
  6. 다른 PC/스마트폰에서 같은 데이터 사용하기
    - HTML 파일을 어느 기기든 열기 → 클라우드 탭에서 같은 URL/Key/워크스페이스 입력 → "⬇️ 지금 다운로드" → 동일 데이터 표시

🛡️ 3중 안전 보존 구조

  • 1차: 브라우저 localStorage — 모든 입력은 즉시 로컬에 저장 (오프라인에서도 작동)
  • 2차: 클라우드 DB(Supabase) — 저장 직후 자동으로 클라우드에 동기화. 브라우저/PC가 망가져도 안전
  • 3차: JSON 백업 파일 — 설정 탭에서 주기적으로 다운로드 → 구글드라이브/원드라이브에 보관 (Supabase가 점검/장애일 때 대비)

이 3가지 중 하나만 살아 있어도 데이터를 100% 복구할 수 있습니다.
